# Definition of Done

Sebuah sprint dianggap selesai apabila:

- Feature selesai dikembangkan
- Tidak ada error
- Mengikuti seluruh dokumen docs/
- Mengikuti Coding Standards
- Sudah diuji
- Tidak merusak fitur lain
- Dokumentasi diperbarui
- Lulus Code Review

---

# AI Development Workflow

Sebelum mengerjakan fitur baru, AI wajib:

1. Membaca seluruh folder docs/.
2. Memastikan implementasi sesuai Business Rules.
3. Memastikan struktur mengikuti Coding Standards.
4. Memastikan API sesuai API Specification.
5. Menghindari duplicate code.
6. Menggunakan reusable component.
7. Tidak mengubah struktur project tanpa alasan yang jelas.
8. Menambahkan atau memperbarui dokumentasi jika ada perubahan desain.

Apabila requirement tidak jelas atau bertentangan dengan dokumentasi, AI harus menghentikan implementasi dan meminta klarifikasi terlebih dahulu.
